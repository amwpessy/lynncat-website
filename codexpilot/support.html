Help for the Codex cockpit.Codex 驾驶舱使用支持。

CodexPilot is built for Mac users who keep Codex running for real work and need visibility into quota, disk read/write pressure, cache growth, and overall Mac health.CodexPilot 专为在 Mac 上长期使用 Codex 的用户设计，用于持续观察额度变化、硬盘读写压力、缓存增长和整机健康状态。

Why is quota unavailable?为什么额度显示不可用？

Quota depends on local Codex data. Open Codex, make sure the account is signed in, then authorize the relevant local log folder in CodexPilot if prompted.额度依赖 Codex 本地数据。请先打开 Codex 并确认账号已登录；如果 CodexPilot 提示授权日志目录，请授权对应目录。

What does disk activity mean?硬盘读写代表什么？

The app tracks read/write movement after launch so heavy Codex sessions can be correlated with disk pressure and cache growth.应用会从启动后开始统计读写变化，帮助把高强度 Codex 会话与硬盘压力、缓存增长联系起来观察。

What does cache cleanup remove?缓存清理会删除什么？

Cleanup targets cache-like runtime files identified by the app. Review the prompt before confirming cleanup, especially if you use custom Codex paths.清理功能面向应用识别出的运行缓存类文件。执行前请查看确认提示，尤其是你使用自定义 Codex 路径时。

Normal and Sport modesNormal 与 Sport 模式

Normal mode uses a calmer refresh rhythm. Sport mode refreshes faster for active debugging or high-load Codex sessions.Normal 模式采用更平稳的刷新节奏；Sport 模式刷新更快，适合正在调试或高负载使用 Codex 时查看。
