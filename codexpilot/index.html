Signals Codex power users need close at hand.Codex 重度用户需要随时看见的关键状态。

CodexPilot focuses on the practical questions that show up during long Codex sessions: quota, disk behavior, cache growth, and Mac performance.长时间运行 Codex 时，CodexPilot 重点关注几个实际问题：额度、硬盘行为、缓存增长和 Mac 性能。

▣

Quota tracking额度跟踪

Reads local Codex session events when quota signals are available, then surfaces remaining quota in the dashboard, menu bar, and Touch Bar.在本地 Codex session 事件可用时读取额度信号，并在主界面、菜单栏和 Touch Bar 中显示剩余额度。

⇄

Disk read/write activity硬盘读写活动

Shows live disk read/write rates and since-launch totals, helping you notice frequent local disk activity while Codex is working.显示实时硬盘读写速率和本次启动后的累计读写量，帮助你观察 Codex 工作时的本地硬盘活动。

⌫

Cache cleanup缓存清理

Estimates common user cache locations and provides a dedicated clean button for cache generated during local development and Codex runs.估算常见用户缓存目录，并提供独立清理按钮，用于清理本地开发和 Codex 运行产生的缓存。

Normal and Sport modesNormal 与 Sport 模式

Use a calm light dashboard for routine monitoring, or switch to a one-second Sport cockpit with gauge-style status cards.日常使用浅色 Normal 仪表盘；需要高频观察时，切换到 1 秒刷新的 Sport 码表模式。

Quota depends on local Codex events.额度显示取决于本地 Codex 事件。

CodexPilot reads quota-related signals from local Codex session logs when those events are available. If the installed Codex version does not write quota information locally, the app shows quota as not reported.CodexPilot 会在本地 Codex session 日志存在额度事件时读取额度相关信号。如果当前 Codex 版本没有把额度信息写入本地日志，应用会显示额度未报告。
