# 三国人物深度评析

> 史实与演义对照，力求还原历史本貌

一个基于《三国志》正史与《三国演义》对比的三国人物深度评析网站，涵盖100位三国历史人物的生平、战绩与评价。

## 📖 项目简介

本网站收录了100位三国时期的重要历史人物，按四大阵营分类：

| 阵营 | 人数 | 说明 |
|------|------|------|
| 蜀汉 | 25 | 刘备建立的季汉政权，以兴复汉室为旗帜 |
| 曹魏 | 25 | 曹操奠基、曹丕代汉，三国实力最强的政权 |
| 东吴 | 25 | 孙氏父子三代经营，割据江东的孙吴政权 |
| 群雄 | 25 | 汉末乱世中未能建立政权的诸侯豪强 |

## ✨ 功能特性

- **人物筛选**：按阵营（全部/蜀汉/曹魏/东吴/群雄）筛选人物
- **深度解析**：每位人物包含500-1500字的详细分析，涵盖生平、战绩、性格评价
- **史实与演义对比**：标注《三国演义》的虚构情节与《三国志》正史记载的差异
- **人物导航**：上一人/下一人快捷浏览
- **暗黑模式**：支持明暗主题切换
- **响应式设计**：适配桌面和移动端

## 🎨 内容特点

每位人物包含：

- **基本信息**：姓名、字号、生卒年、官职
- **标签**：核心关键词
- **深度分析**：4-7个专题小节，基于《三国志》等正史
- **史实与演义**：正史记载与小说虚构的对比
- **综合评价**：对人物的历史定位与评价

## 📚 参考史料

- **正史**：《三国志》《资治通鉴》《后汉书》《华阳国志》《魏书》《魏略》
- **文学**：《三国演义》《中国小说史略》

## 🚀 使用方法

### 直接打开

下载项目后，用浏览器打开 `index.html` 即可，无需安装任何依赖。

### 本地服务器

```bash
# Python
python -m http.server 8000

# Node.js
npx serve
```

然后访问 `http://localhost:8000`
